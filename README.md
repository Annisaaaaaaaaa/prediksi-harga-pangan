## Deskripsi Proyek
Proyek ini bertujuan untuk menganalisis **tren harga pangan nasional** berdasarkan
data historis bulanan serta membangun **model Machine Learning** untuk
memprediksi harga **Beras Premium** pada periode tertentu.

Analisis data disajikan dalam bentuk **dashboard interaktif** yang memungkinkan
pengguna melakukan eksplorasi data dan prediksi harga secara mudah.

## Dataset
Dataset yang digunakan merupakan data harga pangan bulanan nasional Indonesia
yang telah melalui proses **data cleaning dan preprocessing**.

Kolom utama:
- Komoditas
- Tahun
- Bulan
- Harga

## Tools & Teknologi
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Streamlit

## Model Machine Learning
- Jenis Model: **Regresi Linear**
- Target Prediksi: Harga Beras Premium
- Model disimpan dalam format `.pkl`
- Model di-deploy ke dalam aplikasi Streamlit

## Fitur Dashboard
- Filter komoditas pangan
- Filter tahun
- Visualisasi tren harga bulanan
- Input bulan dan tahun untuk prediksi harga Beras Premium
